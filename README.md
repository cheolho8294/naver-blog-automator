# 네이버 블로그 자동 작성

세차·디테일링 전문가를 위한 AI 블로그 글 생성기. 주제와 사진을 넣으면 네이버 블로그 스타일의 SEO 최적화 글을 자동으로 만들어줍니다.

## 주요 기능

- 숫자+경험+솔직함 패턴의 검색 최적화 제목
- 소제목1~4 구조, 1500자 이상, 짧은 문단
- 이전 제품/방법 vs 이번 제품/방법 비교 섹션
- 아쉬웠던 점 3가지 솔직 후기
- 댓글 유도, 저장 권유 CTA 포함
- 모바일 카메라로 직접 촬영 → 바로 글 생성
- PWA 지원 (폰 홈화면 추가)

## 빠른 시작

```bash
# 의존성 설치 (이미 되어있으면 생략)
npm install

# 환경변수 설정
cp .env.example .env.local
# .env.local 파일을 열어서 ANTHROPIC_API_KEY 입력

# 개발 서버 실행
npm run dev
```

브라우저에서 http://localhost:3000 열기.
같은 WiFi의 폰에서는 http://[PC의_IP주소]:3000 으로 접속.

## 배포 (Vercel)

```bash
npm install -g vercel
vercel deploy
```

Vercel 대시보드에서 환경변수 설정:
- `ANTHROPIC_API_KEY` (필수)
- `BLOB_READ_WRITE_TOKEN` (이미지 업로드 시 필요)

## 사용 방법

1. 주제 입력 (예: "세라믹 코팅 시공 후기")
2. 핵심 키워드 입력 (예: "세라믹코팅, 광택")
3. 내 경험 메모 자유롭게 작성
4. 비교 대상 입력 (이전 제품 vs 이번 제품)
5. 사진/영상 업로드 (각 사진마다 설명 추가 가능)
6. "블로그 글 생성하기" 클릭
7. 생성된 HTML을 네이버 블로그 에디터에 붙여넣기
8. [📷 사진N 삽입] 위치에 실제 사진을 직접 추가
